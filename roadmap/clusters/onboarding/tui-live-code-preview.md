# TuiLiveCodePreview

**Cluster:** [onboarding](README.md) · **Status:** shipped 2026-04-15 ·
**Phase:** 2 (unblocked) · **Severity:** n/a

## What it is

Render positive and negative Kotlin fixture code inline as the user
evaluates each controversial rule. Fixtures sourced from the paths
in `controversial-rules.json`.

## Shape

Split view: question + rationale on the left, syntax-highlighted
fixture code on the right. Toggle between "triggers" and "does not
trigger" tabs.
