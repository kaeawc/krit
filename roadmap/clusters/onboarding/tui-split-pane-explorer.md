# TuiSplitPaneExplorer

**Cluster:** [onboarding](README.md) · **Status:** shipped 2026-04-15 ·
**Phase:** 2 (unblocked) · **Severity:** n/a

## What it is

Full-screen split-pane rule explorer: left pane lists all rules with
toggle state and finding count from the target codebase, right pane
shows the selected rule's description + fixture code + sample findings
from the scan. Toggling a rule in the left pane instantly updates
the finding count and removes/adds samples in the right pane.
